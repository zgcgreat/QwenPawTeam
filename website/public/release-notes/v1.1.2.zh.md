## ✨ 新增功能

**Agent 系统**

- **Mission 模式**：通过 `/mission` 命令启动自主多阶段任务执行，Agent 自动规划、执行并迭代修正。支持 `/mission status` 和 `/mission list` 实时查看任务进度 ([#3364](https://github.com/agentscope-ai/QwenPaw/pull/3364), [#3480](https://github.com/agentscope-ai/QwenPaw/pull/3480))
- **文本回复自动重试**：开启后，若模型仅返回文本而未调用工具，Agent 会自动追加提示并重试最多两轮推理 ([#3107](https://github.com/agentscope-ai/QwenPaw/pull/3107))
- **自定义 Agent ID**：在控制台或 API 创建 Agent 时可指定自定义 ID ([#3333](https://github.com/agentscope-ai/QwenPaw/pull/3333))
- **ACP 外部 Agent 委托**：通过 `delegate_external_agent` 工具将任务委托给外部编程 Agent（OpenCode、Qwen、Claude Code、Codex），支持权限管控和实时流式输出 ([#3340](https://github.com/agentscope-ai/QwenPaw/pull/3340))
- **Agent CLI 创建**：通过 `qwenpaw agents create` 命令行创建 Agent，支持模板选择（`default`、`local`、`qa`）和工作区初始化 ([#3385](https://github.com/agentscope-ai/QwenPaw/pull/3385))

**CLI**

- **`qwenpaw doctor`**：新增诊断命令，可检查运行环境、配置、供应商、频道、技能、MCP、记忆、安全等，`doctor fix` 可自动修复常见问题 ([#3371](https://github.com/agentscope-ai/QwenPaw/pull/3371))
- **`qwenpaw skills info`**：通过命令行查看技能详情（启用状态、频道、路径、描述），同时在聊天中新增 `/skills` 斜杠命令 ([#3459](https://github.com/agentscope-ai/QwenPaw/pull/3459))

**记忆**

- **记忆整理（Dream）**：定时运行"Dream"Agent 自动去重和整理 `MEMORY.md` 长期记忆 ([#2177](https://github.com/agentscope-ai/QwenPaw/pull/2177))
- **递归文件监听**：新增 `recursive_file_watcher` 选项，支持将子目录文件纳入记忆索引 ([#3347](https://github.com/agentscope-ai/QwenPaw/pull/3347))

**控制台与界面**

- **技能导入中心**：重新设计技能导入弹窗，支持 URL 校验、技能市场集成，导入冲突时提供覆盖确认 ([#2412](https://github.com/agentscope-ai/QwenPaw/pull/2412), [#3415](https://github.com/agentscope-ai/QwenPaw/pull/3415), [#3482](https://github.com/agentscope-ai/QwenPaw/pull/3482))
- **调试页面**：设置中新增调试页面，可实时查看后端日志 ([#3478](https://github.com/agentscope-ai/QwenPaw/pull/3478), [#3511](https://github.com/agentscope-ai/QwenPaw/pull/3511))

**频道（Channels）**

- **微信引用消息**：支持解析微信引用/回复消息，涵盖文本、图片、语音、文件和视频类型 ([#3483](https://github.com/agentscope-ai/QwenPaw/pull/3483))

## 🔄 变更

- **供应商排序优化**：设置页中供应商列表现按可用性排序，已配置且可用的供应商优先展示 ([#3458](https://github.com/agentscope-ai/QwenPaw/pull/3458))
- **Agent 通信工具拆分**：Agent 间通信工具拆分为同步的 `chat_with_agent` 和异步的 `submit_to_agent` / `check_agent_task`，后台任务提交和查询更加清晰 ([#3485](https://github.com/agentscope-ai/QwenPaw/pull/3485))
